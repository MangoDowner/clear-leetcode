# 672. 灯泡开关 Ⅱ
垃圾题目

## 前三个灯泡决定了其他灯泡的状态
### 解题思路
假设操作1/2/3/4，分别实现了a/b/c/d次，那么
* 灯泡1状态 = (1 + a + c +d) % 2
* 灯泡2状态 = (1 + a + b) % 2
* 灯泡3状态 = (1 + a + c) % 2
* 灯泡4状态 = (1 + a + b +d) % 2

* 1）当``灯泡1==灯泡3``时候，那么``d``为偶数，则``灯泡2==灯泡4``
* 2）当``灯泡1!=灯泡3``时候，那么``d``为奇数，则``灯泡2!=灯泡4``

也就是说``灯泡1/3``，决定了``灯泡4``，即前三个灯泡可以决定所有灯泡状态

### 代码
```golang
func flipLights(n int, m int) int {
	switch n {
	case 0:
		return 0
	case 1:
		return min(2, 1+m)
	case 2:
		return min(4,1+m*2)
	default:
		return min(8, 1+m*3)
	}
}
```
