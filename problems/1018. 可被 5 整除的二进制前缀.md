# 位运算，记得防溢出！
### 解题思路
* 1、 遍历``A``，用``num``记录``A``当前组成的数字
* 2、 依次左移``num``并加上当前的数字，得到新的``num``，不过``num``能整除5，得到true
* 3、 数字是否能被5整除，说明最后一位数字得是``5``或者``10``，
5和10左移2之后，都会得到10的倍数，因此我们可以将``num``对10取余，

这样一来，可以防止溢出。

### 代码

```golang
func prefixesDivBy5(A []int) []bool {
	num, result := 0, make([]bool, len(A))
	for k, v := range A {
		num = num << 1 + v
		if num % 5 == 0 {
			result[k] = true
		}
		num %= 10

	}
	return result
}
```
