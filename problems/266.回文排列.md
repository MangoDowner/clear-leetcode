# 双百哈希法
> 原题链接：[266. 回文排列](https://leetcode-cn.com/problems/palindrome-permutation/)

### 解题思路
能够重构成回文，只要字符成双成对，剩下的字符不超过一个就好了。

### 代码

```golang
func canPermutePalindrome(s string) bool {
	m := make(map[int32]int)
	for _, v := range s {
		if _, ok := m[v]; ok {
			delete(m, v)
		} else {
			m[v] = 0
		}
	}
	return len(m) <= 1
}
```
