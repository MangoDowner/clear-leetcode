> 原题链接：[100. 相同的树](https://leetcode-cn.com/problems/same-tree/)

### 解题思路
* 1、如果s/t两者都是nil，相同
* 2、如果s/t两者只有一个nil，不同
* 3、如果s/t两者的值相同，还要看，他们的左右节点是不是相同
### 代码
```golang
func isSameTree(s *TreeNode, t *TreeNode) bool {
	if s == nil {
		return t == nil
	}
	if t == nil {
		return false
	}
	return s.Val == t.Val && isSameTree(s.Left, t.Left) && isSameTree(s.Right, t.Right)
}
```
