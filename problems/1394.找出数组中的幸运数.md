# 哈希表计算出现次数
## 哈希表法
### 解题思路
* 1、用哈希表``m``，记录每个数字的出现次数
* 2、遍历``m``发现数字和出现次数相同的，再看看其是不是当前最大的，如果是更新最大值
### 代码
```golang
func findLucky(arr []int) int {
	m := make(map[int]int)
	for _, v := range arr {
		m[v]++
	}
	result := -1
	for k, v := range m {
		if k == v && v > result {
			result = k
		}
	}
	return result
}
```
