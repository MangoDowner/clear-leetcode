## 哈希表法
### 代码
```golang
func missingNumber(nums []int) int {
	arr := make([]int, len(nums)+1)
	for _, v := range nums {
		arr[v] = 1
	}
	for k, v := range arr {
		if v == 0 {
			return k
		}
	}
	return 0
}
```
## 累加法
### 解题思路
本来该有的所有数字加起来，减去所有存在的数字，就是那个缺失的数字 
### 代码

```golang
func missingNumber(nums []int) int {
	result := len(nums)
	for k, v := range nums {
		result += k - v
	}
	return result
}
```
## 位运算法
### 解题思路
* 1、一个数字异或自己等于0， a^a=0
* 2、任意数字异或``0``等于其本身,a^0=a
### 代码
```golang
func missingNumber(nums []int) int {
	result := len(nums)
	for k, v := range nums {
		result ^= k ^ v
	}
	return result
}
```
## 排序后查看缺失数字
### 解题思路
排序后，对应索引处的数字如果和索引不一样，那个索引就是缺失的数字
### 代码
```golang
func missingNumber(nums []int) int {
	sort.Ints(nums)
	for k, v := range nums {
		if k != v {
			return k
		}
	}
	return len(nums)
}
```
