# 取和差异=交换数差乘以二
### 解题思路
* 1、分别求``A``/``B``中所有数的和，得到``sumA``/``sumB``
* 2、求``sumA``/``sumB``的差异，得到``dis``
* 3、如果``A``/``B``中有这样两个数，交换后，使得差异``dis``被平掉，就可以满足条件

假设交换的两个数的为``a``/``b``，差为``d``，那么说明``A``/``B``中一个总和减少了``d``，一个增加了``d``，
那么此消彼长，这一次交换，使得两者的和差异减少了``d * 2``。

这样一来，可以得到:
```
d * 2 = dis
```
也就是
```
(a - b) * 2 = dis
即
a - b = dis * 2 
```

### 代码
```golang
func fairCandySwap(A []int, B []int) []int {
	sumA, sumB := 0, 0
	mA, mB := make(map[int]int), make(map[int]int)
	for _, v := range A {
		mA[v] = 1
		sumA += v
	}
	for _, v := range B {
		mB[v] = 1
		sumB += v
	}
	dis := sumA - sumB
	if dis%2 != 0 {
		return []int{}
	}
	for _, a := range A {
		b := a - dis/2
		if mB[b] != 0 {
			return []int{a, b}
		}
		b = a + dis
	}
	return []int{}
}
```
