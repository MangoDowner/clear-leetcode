# 217. 存在重复元素

> 原题链接：[217. 存在重复元素](https://leetcode-cn.com/problems/contains-duplicate/)
## 哈希表维护上次相同数字出现的位置
### 解题思路
* 1、维护一个哈希表``m``，存储每个数字上次出现的位置
* 2、遍历``nums``
* 3、如果``nums[i]``这个数字，上次出现过,可以直接返回true
* 4、遍历结束后，还没出现3中的情况，那么就返回false吧
### 代码

```go
func containsDuplicate(nums []int) bool {
	m := make(map[int]int)
	for _, v := range nums {
		if _, ok := m[v]; ok {
			return true
		}
		m[v] = 0
	}
	return false
}
```
