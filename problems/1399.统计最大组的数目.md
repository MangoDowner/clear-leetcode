# 先求出各种和的计数，再求多少个并列第一

> 原题链接：[1399. 统计最大组的数目](https://leetcode-cn.com/problems/count-largest-group/)


### 解题思路
* 1、先求出各种和的计数，记录为m
* 2、记录最大的计数max
* 3、遍历计数数组m
* 4、找出和max一样的和有多少

### 代码

```golang
func countLargestGroup(n int) int {
	m, max, sum, res := make(map[int]int), 0, 0, 0
	for i := 1; i <= n; i++ {
		sum = getAllNumSum(i)
		m[sum]++
		if m[sum] > max {
			max = m[sum]
		}
	}
	for _, v := range m {
		if v == max {
			res++
		}
	}
	return res
}

func getAllNumSum(n int) int {
	sum := 0
	for n != 0 {
		sum += n % 10
		n /= 10
	}
	return sum
}
```
