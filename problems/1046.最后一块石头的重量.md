# 排序后删除后两位。加入其差值
### 解题思路
1.将``stones``数组从小到大排序
2.求最后两个数字之差为``cha``
3.删除``stones``数组最后两个数字
4.将``cha``加到``stones``数组中
5.重复1-4过程，直到数组只剩下一个元素
### 代码

```go
func lastStoneWeight(stones []int) int {
	for len(stones) > 1 {
		sort.Ints(stones)
		l := len(stones)
		stones = append(stones[:l-2], stones[l-1]-stones[l-2])
	}
	return stones[0]
}
```
