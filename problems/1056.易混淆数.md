# 哈希＋取余法
### 解题思路
* 1.维护哈希表``m``，存储旋转后有新数的数字
* 2.从地位到高位，依次取出``N``每一位数:
    * 1）如果没有``m``中对应项，返回``false``
    * 2）否则，将这个数字组装到新数字``num``里面去，方法是:
```
num = num*10 + 当前数字
```
### 代码

```golang
func confusingNumber(N int) bool {
	m := map[int]int{
		0: 0, 1: 1, 6: 9, 8: 8, 9: 6,
	}
	num, tempN := 0, N
	for tempN != 0 {
		if value, ok := m[tempN%10]; ok {
			num = num*10 + value
			tempN /= 10
		} else {
			return false
		}
	}
	return num != N
}
```
