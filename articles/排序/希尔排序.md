# 希尔排序
## 思路
基于插入排序，先分段排序，所分段越来越小，保持局部有序，然后实行插入排序

希尔排序也称为“缩小增量排序”，它的基本原理如下：
* 1、首先将待排序的数组元素分成多个子序列，使得每个子序列的元素个数相对较少
* 2、然后对各个子序列分别进行直接插入排序
* 3、待整个待排序序列“基本有序”后最后再对所有元素进行一次直接插入排序。
## 代码
```go
func shellSort(a []int) {
	N, h := len(a), 1
	for h < N/3 {
		h = h*3 + 1
	}
	for h >= 1 {
		for i := h; i < N; i++ {
			for j := i; j >= h && a[j] < a[j-h]; j -= h {
				a[j], a[j-h] = a[j-h], a[j]
			}
		}
		h /= 3
	}
}
```
